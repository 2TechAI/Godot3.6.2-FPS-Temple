# Godot3.6.2-FPS-Temple
Godot3.6.2 FPS Temple.MIT license.You can use it make some FPS game or just play it.

# Step 1
Download Godot3.6.2 From:
https://godotengine.org/download/3.x/windows/

# Step 2
Open the editor and import this Temple.
Run it.
You can use Cursor,Codex,Claude Code......to edit this temple.
Or just play this temple.


# So,make your own game is so easy!
